# Task 6: Assess APOEMA Graphics Utility and Assertiveness

## Description

1. **Avaliação Crítica de Pertinência:** Determine a utilidade prática do dashboard para a área específica. O foco é se o gráfico agrega valor estratégico para o coordenador do PPG.
2. **Conflito de Domínio:** Aponte se o DNA de "Ciência da Computação" do APOEMA está prejudicando a narrativa da área atual.
3. **Julgamento de Valor:** Atribua um Score de 1 a 10 para cada gráfico baseado em Relevância, Clareza e Poder de Decisão.
4. **Plano de Ação:** Consolide recomendações de "Manter", "Adaptar" ou "Substituir" de forma puramente textual.
